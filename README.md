w-a-s-d-macro

A simple AutoHotkey macro that automatically presses W, A, S, D keys in a loop with a small delay.

⚙️ Requirements

You need AutoHotkey installed to run this script.

🚀 Features
Automatically cycles through W → A → S → D
Adjustable delay (currently 60ms per key)
Lightweight and easy to use
Start/Stop control with hotkeys
🎮 Controls
R Key: Starts the macro
I Key: Stops the macro
F12 Key: Closes the script completely
🔧 Customization

You can easily change the keys and delay inside the script.


R::  ; Start Key
I::  ; Stop Key
F12::ExitApp


Sleep, 60  ; Key press delay (ms)

⚠️ Warning
Using macros in online games may violate game rules and can result in penalties or bans.
Use responsibly and at your own risk.
